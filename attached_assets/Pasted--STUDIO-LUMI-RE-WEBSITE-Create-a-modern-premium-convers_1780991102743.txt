# STUDIO LUMIÈRE WEBSITE

Create a modern, premium, conversion-focused website for Studio Lumière, a South African web design agency helping businesses launch, grow, and maintain a professional online presence.

The website should feel trustworthy, modern, premium, and business-focused. Every section should communicate one message:

"We make it easy and affordable for businesses to get online."

The website must offer two ways to work with Studio Lumière:

1. Purchase a website outright.
2. Rent a website through an affordable monthly plan.

The goal is to remove barriers that prevent small businesses from getting a professional website.
